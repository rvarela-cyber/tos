How the implementation works: Every CTA button on the live site passes a ?from= URL parameter that maps to a context object in the quote page's JavaScript. On page load, the script reads the parameter, swaps the hero copy, preselects the service dropdown, and renders the appropriate conditional fields. Same form, same backend, same lead routing — fourteen contextual experiences.

Adding a new context requires only an entry in the CONTEXTS and CONDITIONAL_FIELDS objects. No new pages, no duplicate forms.
