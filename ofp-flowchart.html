Key Functions
Wayfinder Popup
Site-wide popup that fires after the user has been on site for one minute. Promotes the Furniture Wayfinder tool and routes the user into the guided discovery flow.
Conditional Quote Forms
Single quote form that adapts based on entry point. One form handles every service, with the launcher dynamically swapping in the right fields whenever a quote button is clicked from a specific service page.
Taxonomy Collaboration Doc
Shared working document for the OFP taxonomy framework. This is the starting point for ongoing collaboration between Mindtap, Springbolt and TOS teams as we finalize categories, filters, and product attribution.
Mobile Menu
Dedicated mobile navigation built to mirror the desktop mega menu while collapsing into a tap-friendly drawer. Optimized for touch targets, fast scroll, and clean section grouping. Loads independently from the desktop menu.
